# GRADALIS 
***

Данный продукт является разработкой для бизнеса. Проект служит для обеспечения сбора и первичной обработки исходной информации, 
необходимой для подготовки отчетности по показателям деятельности, а также для создания единой системы отчетности по данным 
о закупках-продажах оборудования.  

## Возможности, доступные пользователю  
Пользователь получает удобную навигацию по упорядоченным данным, сводки информации по выбранным поставщикам/товарам/клиентам/проектам 
за определенный период времени.  

Пользователь имеет возможность рассчитать доходы и расходы компании, сумму оплаты счета с учетом актуального курса валюты.  

В БД хранятся все необходимые данные о клиентах, компаниях, поставщиках, производителях, товарах, проектах. При построении графиков, 
диаграмм и гистограмм все суммы конвертируются в валюту, запрошенную пользователем, и на основании соотношения этих сумм выполняется 
построение графиков.  

## Средства, используемые для реализации  
Для реализации используется язык C++ с различными расширениями для него (Qt для реализации графического оконного интерфейса, libpqxx (в составе драйверов qt) для работы с базами данных). 
В качестве СУБД - PostgreSQL.  

# Сборка проекта
Для сборки проекта вам понадобится сборка Qt от 5.8 с QtCharts, QtSql. Сборка PostgresSQL 9.6 и более. Важно, что разрядность и компилятор(!) сборки должны совпадать с выбранными вами для сборки разрядностью и компилятором соответсвенно. Особенно это проблематично для сборки на Windows, поэтому настоятельно советуем использовать для этого VC++ компилятор, добавленный(скорее всего вручную) в Qt Creator. В .pro файл следует указать путь до `/include` и `libpq.lib` вашей инсталляции PostgreSQL.
```
    #INCLUDEPATH += \
    # "C:\postgres32\include"
    
    #LIBS += \
    # "C:\postgres32\lib\libpq.lib"

В свойствах проекта также нужно добавить в `PATH:` пути до `/include/` и `/bin/` инсталляции PostgreSQL

## Linux
Для линукс достаточно установить правильный пакет с PostgreSQL. Однако это облегчает только динамическую линковку.
